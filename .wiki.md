# Project Summary
The project is a premium institutional website for SoulPellegriniTech, a strategic technology consulting firm specializing in digital transformation, artificial intelligence, governance, and cybersecurity. The website enhances client interactions for healthcare executives by providing a modern interface that effectively showcases services and successful case studies. It features updated branding with a new logo, optimized for web performance, and improved navigation elements for an enhanced user experience.

# Project Module Description
The project consists of several functional modules:
- **Home**: Introduction and overview of services.
- **About**: Company mission, vision, values, and timeline focused on strategic IT consulting and digital transformation.
- **Services**: Detailed descriptions of offered services, including Consulting, AI, Cybersecurity, and Digital Transformation.
- **Advisory**: Information on advisory programs for executives across various sectors.
- **Cases**: Showcase of successful projects with anonymized testimonials.
- **Contact**: Contact form and company information, integrated with Formspree for email submissions.
- **WhatsApp Button**: A floating button for direct communication, featuring animation and tooltip for user interaction.
- **Branding**: Updated logo used in the Navbar, Footer, and favicon, optimized for web use.

# File Description Inventory
- **README.md**: Project overview and setup instructions.
- **.npmrc**: Configuration file to enforce legacy peer dependencies.
- **index.html**: Main HTML file for the application, updated to include the new favicon.
- **package.json**: Project dependencies and scripts.
- **package-lock.json**: Automatically generated file that locks the versions of dependencies.
- **src/App.tsx**: Main application component that defines routes and includes the updated Navbar with the new logo.
- **src/assets**: Contains the optimized logo image now imported as a module.
- **src/pages**: Contains all page components for the website.
- **src/components**: Contains reusable UI components, including the updated Navbar and Footer with the new logo references.
- **src/hooks**: Custom hooks for managing state and logic.
- **src/lib**: Utility functions and API handling; `api.ts` is now a placeholder.
- **public/**: Static assets like images and icons, including the optimized favicon and logo.
- **vercel.json**: Configuration for deploying the project to Vercel.

# Usage
To set up the project:
1. Install dependencies: `npm install --legacy-peer-deps`
2. Build the project: `npm run build`
3. Run the development server: `npm run dev`

### Deploying to Vercel
1. Ensure that the `.npmrc` file is present in both the project root and `app/frontend/` to manage dependencies.
2. Set the **Root Directory** to `app/frontend` in the Vercel Dashboard under Settings в†’ General.
3. Push changes to GitHub to trigger redeployment.

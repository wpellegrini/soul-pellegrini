# SOUL PELLEGRINI TECH - Premium Institutional Website

## Design Guidelines

### Design References
- **McKinsey.com**: Clean executive authority, premium consulting feel
- **Accenture.com**: Bold tech-forward design, modern corporate
- **Style**: Dark Executive Premium + Green Accent + Modern Minimalism

### Color Palette
- Primary Dark: #0A0F0D (Near Black with green tint - main background)
- Secondary Dark: #111916 (Dark green-black - cards/sections)
- Accent Green: #00C853 (Vibrant green - CTAs, highlights)
- Dark Green: #1B5E20 (Deep green - secondary accents)
- Gold: #C9A84C (Gold - premium details, borders)
- White: #FFFFFF (Text primary)
- Light Gray: #B0BEC5 (Text secondary)
- Surface: #1A2420 (Card backgrounds)

### Typography
- Headings: Inter font-weight 700/600
- Body: Inter font-weight 400
- Navigation: Inter font-weight 600 uppercase tracking-wider

### Key Component Styles
- **Buttons**: Green gradient (#00C853 to #1B5E20), white text, rounded-lg, hover: brighten
- **Cards**: Dark surface (#1A2420), 1px border (#2A3A30), rounded-xl, hover: lift with green glow
- **Navigation**: Fixed top, backdrop-blur, transparent to dark on scroll
- **Sections**: Full width, alternating dark backgrounds, 96px vertical padding
